## 目录介绍
- Common:存放自己封装文件
    *basepage: 封装的底层方法 
    *contant: 各类路径
    *my_log: 日志类
    
- Libraries: 存放别人封装好的类
    *ddt: 数据驱动类
    *HTMLTestRunnerNew: HTML测试报告类
    
- Outputs:存放输出的截图、日志和报告
    *imgs: 存放错误截图
    *logs: 存放运行日志
    *reports: 存放测试报告
    
- PageLocators:存放页面元素定位
    *bid_page_locators: 标页面元素定位
    *index_page_locators: 标首页元素定位
    *login_page_locators: 登录页面元素定位
    *user_page_locators: 个人信息页面元素定位
    
- PageObjects:存放功能页面
    *bid_page: 标页面操作
    *index_page: 标首页操作
    *login_page: 登录页面操作
    *user_page: 个人信息页面操作
    
- TestCases:存放测试用例
    *test_invest: 投标测试用例
    *test_login: 登录测试用例
    
- TestDatas:存放测试数据
    *bid_datas: 投标数据
    *common_datas: 公共数据
    *login_datas: 登录数据
